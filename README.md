# VK Авто-отчёт — расширение для Яндекс Браузера

Расширение автоматически отправляет отчёт (текст по шаблону) в выбранный чат
ВКонтакте по расписанию — например, ежедневный рабочий отчёт в 18:00 по будням.
Также можно отправить отчёт вручную кнопкой «Отправить сейчас».

Работает через веб-версию vk.com: в назначенное время расширение открывает
нужный чат в фоновой вкладке, вставляет текст в поле ввода и отправляет его.
Токены и API ВК не используются — нужен только выполненный вход в ВК в браузере.

## Установка в Яндекс Браузере

1. Скачайте/склонируйте эту папку на компьютер.
2. Откройте в браузере адрес `browser://extensions` (или Меню → Дополнения →
   внизу «Каталог расширений» → значок настроек).
3. Включите переключатель **«Режим разработчика»** (справа сверху).
4. Нажмите **«Загрузить распакованное расширение»** и укажите папку проекта
   (ту, где лежит `manifest.json`).
5. Значок расширения появится на панели. Закрепите его для удобства.

## Настройка

Нажмите на значок расширения:

| Поле | Что указывать |
|---|---|
| Чат | id получателя из адресной строки: откройте нужный чат на `vk.com/im` — в адресе будет `?sel=…`. Число — личный диалог, `c15` — беседа №15, можно короткое имя (`durov`) |
| Время отправки | во сколько отправлять отчёт |
| Дни недели | по умолчанию Пн–Пт |
| Шаблон отчёта | текст сообщения; `{date}` заменится на дату, `{time}` — на время |

Включите галочку **«Автоотправка включена»** и нажмите **«Сохранить»**.

## Как происходит отправка

Расширение открывает (или активирует) вкладку с чатом, находит поле ввода
и печатает текст через протокол DevTools (`chrome.debugger`) — для страницы
это неотличимо от настоящего набора с клавиатуры, поэтому способ работает
с любым редактором ВК. На время отправки вверху окна появляется полоска
«Расширение "VK Авто-отчёт" запустило отладку этого браузера» — это
нормально, она исчезает после отправки. Результат (успех или текст ошибки)
показывается системным уведомлением.

## Важно

- В момент отправки браузер должен быть запущен, а в ВК выполнен вход.
- Во время отправки окно с чатом становится активным — не печатайте в этот
  момент в другом поле этого же окна.
- Расширение защищено от дублей: отчёт уходит не более одного раза в день.
- ВК периодически меняет вёрстку сайта. Расширение сначала пробует известные
  селекторы, а если их нет — ищет поле ввода и кнопку «Отправить» эвристически.
  Если отправка всё равно не работает, откройте чат, нажмите F12 → вкладка
  «Консоль» и повторите отправку: расширение пишет туда, какой элемент нашло
  и на каком шаге остановилось. Эти строки помогут поправить селекторы в
  `content.js`.
- После обновления файлов расширения нажмите «Обновить» на
  `browser://extensions` и перезагрузите вкладку с ВК.
- Используйте расширение для своих чатов и с разумной частотой — массовая
  автоматическая рассылка нарушает правила ВКонтакте.

## Структура проекта

```
manifest.json   — манифест расширения (Manifest V3)
background.js   — service worker: расписание (chrome.alarms), открытие чата
content.js      — вставка текста в поле ввода ВК и отправка
popup.html/js/css — окно настроек
icons/          — иконки
```
